### **Lv4**
完成对一个类的属性的赋值

流程为：输入一个对象(类型不确定，但是可以确定有且仅有一个属性，并且这个属性的类型不是int就
是String)，之后判断它的属性是int类型还是String类型，如果是int类型就将“123”转型为整形，然后赋
值给这个属性，反之则直接将String赋值给这个属性

此处需要用到的知识点：
>
>1.利用反射获取类的属性的类型
>
>2.利用反射对对象的属性进行赋值
